Normal distribution (Gaussian distribution)

probability density function (pdf) \[ f(x) = \frac{1}{\sigma \sqrt{2\pi} }e^{ -\frac{(x - \mu)^2}{ 2 \sigma^2} } \]

with mean \(\mu\) and standard deviation \(\sigma\)

Normal distribution

  • a type of bell curve
  • two parameters
    • mean: determines the center of the bell curve
    • variance (and standard deviation \(\sigma\)): determines the width of the bell curve

Normal distribution

  • It is generally observed that the distribution of data is normal.
    • When plotting a graph with the value of the variable on the x-axis and the count of the values on the y-axis, we get a bell shape curve.
    • The center of the curve represents the mean of the data set.
    • Half of values lie to the left of the mean and the other half lie to the right of the graph.

Normal distribution

Empirical rule

  • The probability that a variable will be within +1 or -1 standard deviation of the mean is 0.68 -- i.e., 68% of the values will be within 1 standard deviation (SD) of the mean. The probability that the variable will be on average within 2 and 3 SDs will be 0.95 and 0.997, respectively.
  • Given that 99.7% of the values will fall within 3 standard deviations, we're confidence that value beyond 3 will be highly unlikely.

Modes (peaks)

  • the mode of the data is the peak of the bell
  • unimodal: a distribution with one mode
  • bimodal: a distribution with two modes
  • multimodal: a distribution with two or more modes

symmetric v. skewed

skewed to the right -- more likely to see extreme values to the right of the mode

Power law (highly skewed)

  • a straight-line on the log-log plot: often the signature of a power law
    • However, the straight-line is a necessary but not sufficient condition to say if the data follow a power law

Normality Test in R

  • Many of statistical methods including correlation, regression, t-test, and analysis of variance (ANOVA) assume that the data follows a normal (Gaussian) distribution. These tests are called parametric tests, because their validity depends on the distribution of the data.
  • Normality and the other assumptions made by these tests should be taken seriously to draw reliable interpretation of the analysis.
  • Before using a parametric test, make sure that the test assumptions are met. If the assumptions are violated, non-paramatric tests should be used.

Null hypothesis: The data is normally distributed. If p>0.05, normality can be assumed.

Normality test

  • Visual methods, e.g., density plot and Q-Q plot can be used to inspect normality visually.
    • Q-Q plot (or quantile-quantile plot) draws the correlation between a given sample and the theoretical (normal) distribution. A 45-degree reference line is also plotted in order to see whether the data deviates from normality.
  • There are several methods for normality test, e.g., Kolmogorov-Smirnov (K-S) normality test and Shapiro-Wilk's test (shapiro.test()).
    • The null hypothesis is that sample distribution is normal. If the test is significant, the distribution is non-normal.
    • Shapiro-Wilk's method is more robust and is widely recommended.
    • Both tests are sensitive to outliers and are influenced by sample size. Small samples most often pass normality tests.
    • Better to combine visual inspection and significance test.

Normality test

What if the data is not normally distributed?

  • Some options
    • Transform the variable (repeating the normality checks on the transformed data): Common transformations include taking the log or square root of the variable
    • Use a non-parametric test: Non-parametric tests are often called distribution free tests and can be used instead of their parametric counterpart

Probability distributions in R

Useful R functions:

  • dname calculates density (pdf) at input x.
  • pname calculates distribution (cdf) at input x.
  • qname calculates the quantile at an input probability.
  • rname generates a random draw from a particular distribution.

Note that name is the name of the given distribution.

Probability distributions in R

## To calculate the value of the pdf at x = 3, i.e., the height of the curve at x = 3
dnorm(x = 3, mean = 2, sd = 5)

## [1] 0.07820854

## To calculate the value of the cdf at x = 3, i.e.,  P(X<=3)
pnorm(q = 3, mean = 2, sd = 5)

## [1] 0.5792597

## To calculate the quantile for probability 0.975
qnorm(p = 0.975, mean = 2, sd = 5)

## [1] 11.79982

## To generate a random sample of size n = 10
