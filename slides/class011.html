Install R packages

R has a lot of powerful packages. You need to load a package into memory before using it.

library(ggplot2) ## a package for plotting

Install R packages

If you see an error message:

there is no package called 'ggplot2'

it means you don't have this package and you need to install it.

  • Using GUI
  • Using command

Install R packages (using command)

## install directly from the default repository http://cran.r-project.org/
install.packages('ggplot2') 
## or install a bunch of them
install.packages(c('ggplot2','tidyverse','dplyr'))

Or you can download from the source:

## option 1: download the package source, and then install from the downloaded file 
download.file('http://cran.r-project.org/src/contrib/ggplot2_0.9.3.1.tar.gz',
             destfile='~/Downloads/ggplot2_0.9.3.1.tar.gz')
install.packages('~/Downloads/ggplot2_0.9.3.1.tar.gz',repo=NULL,type='source')
## option 2: install from a source URL
install.packages('http://cran.r-project.org/src/contrib/ggplot2_0.9.3.1.tar.gz',repo=NULL, type="source")

Plotting

## Histogram and Density
library('ggplot2') 
theme_set(theme_bw()) # change the default ggplot theme to black-and-white

ggplot(mydata, aes(x = Height)) +
  geom_histogram(binwidth=1)

Plotting

ggplot(mydata, aes(x = Height)) +
  geom_histogram(binwidth=0.5) # try different value for bandwidth

Plotting

  geom_density()

Plotting

## conditional density plot
ggplot(mydata, aes(x = Weight, fill = Gender)) +
